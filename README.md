# Would You Do?

A phone-first Tinder-style dilemma feed built with React/Vite + Supabase.

## What it does
- Shows one dilemma card at a time
- Swipe/tap Option A or Option B
- Stores anonymous votes by browser/device
- Reveals crowd percentages immediately
- Supports categories, intensity, and monetization-ready content structure

## Deploy stack
- GitHub for code
- Supabase for database/API
- Vercel for hosting

## Environment variables
Set these in Vercel:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
